# Analisis Perbandingan Algoritma Iteratif dan Rekursif: Penghitung Kata
Program ini merupakan implementasi dari tugas besar mata kuliah Analisis Kompleksitas Algoritma yang bertujuan untuk menghitung jumlah kata dalam sebuah kalimat atau pesan menggunakan dua pendekatan algoritma yang berbeda: Iteratif (Looping) dan Rekursif.

# Deskripsi
Dalam pemrosesan teks, menghitung jumlah kata adalah operasi dasar yang penting. program ini membandingkan efisiensi antara metode perulangan biasa dengan pemanggilan fungsi secara berulang (rekursif) dalam menyelesaikan masalah ini.

# Fitur Utama
1. Metode Iteratif (hitungPakaiLooping): Menggunakan for loop untuk menelusuri setiap karakter
   dalam teks dan menghitung kata berdasarkan transisi dari spasi ke karakter. 
2. Metode Rekursif (hitungPakaiRekursif): Menggunakan fungsi yang memanggil dirinya sendiri untuk     memproses karakter satu per satu hingga akhir teks.
